Machine Learning
-------------------
-- Disease Prediction

-- Customer Churn

-- Prediction and Recommendation

1. Supervised Learning
- When we have labeled data
- Data having both features and target
- Output Known
- Salary Prediction, Disease Prediction

	a. Regression
	- If we need to predict or recommend continuous data then use
	  regression.
	- Salary, Whether, Student Marks, ....
	1. Linear Regression - 97%
	2. Polynomial Regression
	3. Random Forest Regressor - 98%
	4. XGBoost Regressor - 99%
	5. Decision Tree Regressor
	6. K-Nearest Regressor
	7. Support Vector Regressor

	b. Classification
	- If we need to predict or recommend class data (yes/no, true/false,
	  0/1, 0/1/2/3/4). Binary class or Multi Class.
	- Disease Prediction, Customer Churn, Students Grade, Employee 	Termination/Active Status, ....

	1. Logistic Regression
	2. KNN (K Nearest Neighbors)
	3. SVM (Support Vector Machine)
	4. Random Forest Classifier
	5. Decision Tree Classifier
	6. XGBoost Classifier


2. Unsupervised Learning
- When we have unlabeled data
- Only features
- Target separate on the basis of Elbow Method

	a. Clustering
	- If we need to segment data into multiple clusters.
	- Bank Loan System, Customer Segmentation
	1. K-Means Clustering
	2. PCA (Principle Component Analysis)
	3. Agglomerative Clustering
	4. Hierarchical Clustering


3. Reinforcement Learning
	- Self Learning Algorithm

Process of Machine Learning
---------------------------------
1. Import Packages (Scikit-Learning)
2. EDA
3. Extract Feature and Target
	- All feature must be independent to each other.
	- Target must be dependent to all features.
4. Train Test Split
	Rule: 80/20 (80% train, 20% Test), 70/30, 60/40, 50/50
5. Feature Scaling
	Standardization
	---------------------
	1. Standard Scaler
	2. Min Max Scaler
	
	Label Encoding
	------------------
	1. Label Encoder
	2. One-Hot Encoder

6. Define Machine Learning Model
7. Fit data into model (Feature Training and Target Training)
8. Predict Data (Feature Testing)
9. Metrics (Accuracy)
10. Visualization
11. New Data Prediction
